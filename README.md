# 🚀 Yulin's Personal Portfolio | 前端開發作品集

## 💡 關於我 (About Me)

你好，我是 **煜琳 (Yulin)**。
擁有食品科學碩士背景，我將實驗室追求「精準」與「再現性」的精神帶入前端開發。我擅長處理複雜的資料流與具備邏輯美感的程式架構。目前正專注於 React 開發，目標是成為一名能兼顧商業邏輯的前端工程師。

這是一個使用 **React** 建立的個人作品集網站，收錄了從基礎切版、動態交互到完整電商邏輯的前端開發歷程。本網站旨在展現我將「食品科學的嚴謹邏輯」轉化為「前端工程實作」的技術成果。

## 🛠️ 技術棧 (Tech Stack)

### Core Framework & Tools
* **Library:** React (Functional Components, Hooks)
* **Build Tool:** Vite
* **Routing:** React Router 6
* **State Management:** Redux Toolkit (RTK)
* **Form Logic:** React Hook Form & Yup 驗證

### Styling & Animation
* **CSS:** SASS/SCSS (7-1 Pattern), Bootstrap 5
* **Animation:** GSAP (ScrollTrigger, SplitText), AOS
* **Components:** Swiper, SweetAlert2, Chart.js

### Development Process
* **Version Control:** Git / GitHub Workflow
* **Code Quality:** ESLint / Prettier
* **API Handling:** Axios / RESTful API Integration

## 📂 專案核心亮點

### 1. Boardreams 桌遊電商平台
* **技術重點：** Redux Toolkit 全域狀態管理、前後台 API 串接、權限控管系統。
* **特色：** 高對比視覺設計，將電商流程遊戲化，提升使用者沉浸感。

* **技術重點：** 高階 GSAP 滾動動畫實作。
